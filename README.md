# Fitworks - Project #2

A responsive fitness studio landing page built with pure HTML and CSS.

## Features
- Fully responsive across mobile and desktop
- Hero image header with navigation
- Exercise gallery with labeled images
- Stats callout section
- Contact form in footer

## Notes
This is a static layout demo. Nav links, the Let's Chat button,
and contact form are not functional — built to demonstrate
responsive design and CSS layout skills only.
